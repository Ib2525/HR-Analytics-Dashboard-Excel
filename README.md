## 🎯 Project Overview
This project focuses on Human Resources analytics for "IAMA Company for Consulting". Developed to transform complex employee data into an interactive dashboard, it provides management with a clear view of workforce demographics, payroll distribution, and departmental headcount. 

## 📊 Key Performance Indicators (KPIs)
- **Total Employees:** 300 👨‍💼
- **Total Salary Run:** $1,336,352 💰
- **Avg. Years of Experience:** 19 Years 🌟

## 🛠️ Technical Skills & Methodologies
- **Data Structuring:** Built a robust relational model using interconnected Pivot Tables.
- **Interactive UI:** Custom-built navigation pane (Report 1 & Report 2) and interactive Slicers (Department, Gender, Country).
- **Advanced Visualizations:** Map charts for geographic distribution, Donut charts for gender/bonus percentages, and clustered column charts.

## 💡 Key Business Insights
1. **Departmental Concentration:** The Manufacturing (71 employees) and Marketing (65 employees) departments hold the largest headcount.
2. **Geographic Footprint:** The majority of the workforce is based in Egypt (99 employees), followed by Saudi Arabia (72 employees).
3. **Gender Demographics:** The workforce consists of 62% Male and 38% Female employees, providing a baseline for diversity metrics.
